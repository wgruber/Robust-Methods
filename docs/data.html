Data

The data used throughout this workshop can be downloaded from the Blackboard. Following data sets are included in the ZIP-File:

  1. FieldLawson2003.csv (Field 2003)
  2. fileKellyetalz.csv (Kelly 2010)
  3. FieldCH2008.csv (Field 2008)
  4. RCTWide.csv

A detailed description of the data can be found in Field & Wilcox (Field 2017), page 24.
